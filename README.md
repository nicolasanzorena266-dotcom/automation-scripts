# Automation Scripts

Small scripts created to automate repetitive operational tasks, organize files, speed up manual workflows, and reduce avoidable errors.

## Purpose

This repository contains practical scripts focused on operational efficiency, especially for process analysis, quality tasks, file handling, and workflow support.

## Examples of use

* Organizing folders and files.
* Cleaning or preparing data.
* Automating repetitive browser or system actions.
* Supporting CX and quality analysis workflows.

## Notes

All examples are anonymized and do not include private, internal, or sensitive information.


## Available scripts

### PowerShell

* **create-working-folders.ps1**
  Creates a basic folder structure with input, output, logs, and documentation folders.

  Useful for preparing a clean workspace before starting an operational analysis task.
